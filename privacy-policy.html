Politique de Confidentialité

Dernière mise à jour : 23 mai 2025

1. Introduction

Cette politique de confidentialité décrit comment TrainSmart AI Coach ("nous", "notre", "l'app") collecte, utilise et protège vos informations personnelles lorsque vous utilisez notre application mobile.

2. Données que nous collectons

2.1 Informations d'authentification

  • Adresse e-mail : Utilisée pour créer et gérer votre compte via Firebase Authentication
  • Identifiants utilisateur : ID unique généré automatiquement pour votre compte

2.2 Données de santé et fitness

  • Données HealthKit : Nombre de pas, calories brûlées, activité physique (avec votre autorisation explicite)
  • Données d'entraînement : Exercices effectués, durée, intensité, historique des séances
  • Objectifs fitness : Buts personnels que vous définissez dans l'app

2.3 Contenu multimédia

  • Enregistrements audio : Commandes vocales pour l'assistant IA (traitées localement et non stockées)

2.4 Données d'utilisation

  • Interactions avec l'app : Utilisation des crédits IA, fonctionnalités utilisées
  • Données de parrainage : Codes de référencement si vous participez au programme

3. Comment nous utilisons vos données

  • Fonctionnement de l'app : Fournir les services d'entraînement personnalisé et d'assistance IA
  • Personnalisation : Adapter les entraînements à vos objectifs et données fitness
  • Synchronisation : Sauvegarder vos données entre vos appareils
  • Support client : Répondre à vos questions et résoudre les problèmes techniques
  • Amélioration du service : Analyser l'utilisation pour améliorer l'app (données agrégées et anonymisées)

4. Partage des données

Nous ne vendons jamais vos données personnelles.

Vos données peuvent être partagées uniquement dans les cas suivants :

  • Services techniques : Firebase (Google) pour l'authentification et stockage sécurisé
  • Obligations légales : Si requis par la loi ou pour protéger nos droits
  • Avec votre consentement : Pour tout autre usage, nous demanderons votre autorisation explicite

5. Stockage et sécurité

  • Chiffrement : Toutes les données sont chiffrées en transit et au repos
  • Stockage : Données stockées sur les serveurs sécurisés de Firebase (Google Cloud)
  • Accès : Accès limité aux données uniquement pour le personnel autorisé
  • Rétention : Vos données sont conservées tant que votre compte est actif

6. Vos droits

Vous avez le droit de :

  • Accéder à vos données personnelles
  • Corriger des informations inexactes
  • Supprimer votre compte et vos données
  • Exporter vos données dans un format portable
  • Retirer votre consentement pour l'accès aux données HealthKit

7. Données des mineurs

Notre application n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations personnelles d'enfants de moins de 13 ans.

8. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité. Toute modification importante vous sera notifiée via l'application ou par e-mail.

9. Conformité réglementaire

Cette politique est conforme au :

  • RGPD (Règlement Général sur la Protection des Données)
  • CCPA (California Consumer Privacy Act)
  • Guidelines Apple pour la confidentialité des applications

10. Contact

Pour toute question concernant cette politique de confidentialité ou vos données personnelles, contactez-nous :

  • E-mail : privacy@trainsmart-ai.com
  • Réponse : Nous nous engageons à répondre dans les 72 heures
